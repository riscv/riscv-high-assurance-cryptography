== Acronyms and Abbreviations

[float="center",align="center",width="100%",cols="<10%,<90%",options="header"]
|===
| Acronym | Description
|  ACE    | Atomic Cryptography Extension
|  AEAD   | Authenticated Encryption with Associated Data
|  AES    | Advanced Encryption Standard
|  ASID   | Application Space Identifier
|  CBC    | Cipher Block Chaining (mode of encryption)
|  CC     | Cryptographic Context
|  CF     | Context File
|  CFB    | Cipher Feedback (mode of encryption)
|  CMAC   | Cipher-based Message Authentication Code
|  CTK    | Context Transport Key
|  CSR    | Control and Status Register
|  CTR    | Counter (mode of encryption)
|  DCTK   | Derived Context Transport Key
|  ECB    | Electronic Codebook (mode of encryption)
|  FIB    | Focused Ion Beam
|  GCM    | Galois Counter Mode
|  GPA    | Guest Physical Address
|  GPR    | General-Purpose Register
|  GVA    | Guest Virtual Address
|  hart   | Hardware Thread
|  ISE    | Instruction Set Extension
|  IV     | Initialization Vector
|  impdef | Implementation-defined
|  HLOS   | High-Level Operating System
|  HV     | Hypervisor
|  KMA    | Key Management Applet
|  LFSR   | Linear Feedback Shift Register (also: LSFR-based mode of operation)
|  LS     | Locality Secrets
|  LST    | Locality Secrets Table
|  MAC    | Message Authentication Code
|  MM     | Machine Mode
|  OFB    | Output Feedback (mode of encryption)
|  OS     | Operating System
|  PA     | Physical Address
|  PCBC   | Propagating Cipher Block Chaining (mode of encryption)
|  PRF    | Pseudo-Random Function
|  PRP    | Pseudo-Random Permutation
|  RCTK   | Root Context Transport Key
|  RUP    | Release of Unencrypted Plaintext attack
|  SCC    | Sealed Cryptographic Context
|  SEM    | Scanning Electron Microscope
|  SIV    | Synthetic Initialization Vector
|  SKT    | System Keys Table
|  TEE    | Trusted Execution Environment
|  VA     | Virtual Address
|  VMID   | Virtual Machine Identifier
|  WARL   | Write Anything Read Legal
|  WARZ   | Write Anything Read-as Zero
|  XCTR   | ``xor``ed Counter (mode of encryption)
|  XEX    | Xor-Encrypt-Xor mode of operation
|  XLFSR  | ``xor``ed Linear Feedback Shift Register (mode of encryption)
|  XOF    | eXtendable Output Function
|  XTS    | XEX mode with cipherText Stealing
|===
