Instructions for mac

https://docs.asciidoctor.org/asciidoctor/latest/install/macos/

Install ruby
for instance, using
    brew install ruby

Make sure following packages are installed:

    fonts-lyx
    ditaa
    pandoc
    graphviz
    ghostscript
    pdf2svg
    pkg-config
    python3
    python3-pip
    nodejs
    texlive-latex-base
    texlive-fonts-recommended
    texlive-fonts-extra
    texlive-latex-extra
    texlive-science
    texlive-luatex

install following gems and dependencies

    asciidoctor
    asciidoctor-sail
    asciidoctor-bibtex
    asciidoctor-diagram
    asciidoctor-lists
    asciidoctor-pdf
    asciidoctor-pdf-mathjax
    asciidoctor-mathematical
    asciidoctor-mathjax3
    asciidoctor-mathml
    asciidoctor-epub3
    asciidoctor-diagram-ditaamini
    asciidoctor-diagram-plantuml
    citeproc-ruby
    coderay
    csl-styles
    json
    mathematical
    pygments.rb
    rghost
    rouge
    write_xlsx

If necessary, build mathematical from sources,
make sure to use --recurse-submodules otherwise some dependencies may not be included and compilation will fail
    gh repo clone gjtorikian/mathematical -- --recurse-submodules
    cd mathematical
    gem build mathematical.gemspec
    gem install mathematical-1.7.0.gem

Other packages may be needed:
    bison
    cmake
    curl
    flex
    git
    default-jre-headless (not on macos)
    libcairo2 (macos: cairo)
    libgdk-pixbuf-2.0-0 (not on macos)
    libpango-1.0-0 (macos: pango)
    libpangocairo-1.0-0 (not on macos)
    libpangoft2-1.0-0
    libxml2 (in some cases libxml2-16)
    libglib2.0-0 (macos: glib)
    libgif7 (not on macos)
    libwebp7 (webp)
    libzstd1 (macos: zstd)
    libjpeg-turbo8 (not on macos)
    librsvg2-2 (not on macos)
    libxft2 (not on macos)

Modify
asciidoctor-diagram/tikz/converter.rb
as follows


require_relative '../diagram_converter'
require_relative '../util/cli_generator'
require_relative '../util/platform'

module Asciidoctor
  module Diagram
    # @private
    class TikZConverter
      include DiagramConverter
      include CliGenerator

      def supported_formats
        [:pdf, :svg]
      end

      def collect_options(source)
        {
          :preamble => source.opt('preamble') || source.attr('preamble') == 'true',
          :command => source.attr('command'),
        }
      end

      def convert(source, format, options)
        if options[:command]
          latexpath = source.find_command(options[:command])
        else
          latexpath = source.find_command('pdflatex')
        end

        if format == :svg
          svgpath = source.find_command('pdf2svg')
        else
          svgpath = nil
        end

        if options[:preamble]
          preamble, body = source.to_s.split(/^~~~~$/, 2)
          unless body
            body = preamble
            preamble = ''
          end
        else
          preamble = ''
          body = source.to_s
        end

        latex = <<'END'
\documentclass[border=2bp, tikz]{standalone}
\usepackage{tikz}
END
        latex << preamble
        latex << <<'END'
\begin{document}
\begingroup
\tikzset{every picture/.style={scale=1}}
END
        latex << body
        latex << <<'END'
\endgroup
\end{document}
END

        pdf = generate_file(latexpath, 'tex', 'pdf', latex) do |tool_path, input_path, output_path|
          {
              :args => [tool_path, '-shell-escape', '-file-line-error', '-interaction=nonstopmode', '-output-directory', Platform.native_path(File.dirname(output_path)), Platform.native_path(input_path)],
              :out_file => "#{File.dirname(input_path)}/#{File.basename(input_path, '.*')}.pdf"
          }
        end

        if svgpath
          generate_file(svgpath, 'pdf', 'svg', pdf) do |tool_path, input_path, output_path|
            {
              :args => [tool_path, Platform.native_path(input_path), Platform.native_path(output_path)],
              :chdir => source.base_dir
            }
          end
        else
          pdf
        end
      end
    end
  end
end
