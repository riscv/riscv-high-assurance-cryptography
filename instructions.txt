Instructions for mac

https://docs.asciidoctor.org/asciidoctor/latest/install/macos/

Install ruby
for instance, using
    brew install ruby

Make sure following packages are installed:

    fonts-lyx
    ditaa
    pandoc
    graphviz
    ghostscript
    pdf2svg
    pkg-config
    python3
    python3-pip
    nodejs
    texlive-latex-base
    texlive-fonts-recommended
    texlive-fonts-extra
    texlive-latex-extra
    texlive-science
    texlive-luatex

install following gems and dependencies

gem install    asciidoctor
gem install    asciidoctor-pdf
gem install    asciidoctor-sail
gem install    asciidoctor-bibtex
gem install    asciidoctor-diagram
gem install    asciidoctor-lists
gem install    asciidoctor-pdf-mathjax
gem install    asciidoctor-mathjax3
gem install    asciidoctor-mathml
gem install    asciidoctor-epub3
gem install    asciidoctor-diagram-ditaamini
gem install    asciidoctor-diagram-plantuml
gem install    citeproc-ruby
gem install    coderay
gem install    csl-styles
gem install    json
gem install    pygments.rb
gem install    rghost
gem install    rouge
gem install    write_xlsx


gem install    mathematical
gem install    asciidoctor-mathematical


If necessary, build mathematical from sources,
make sure to use --recurse-submodules otherwise some dependencies may not be included and compilation will fail

    gh repo clone gjtorikian/mathematical -- --recurse-submodules
    cd mathematical
    gem build mathematical.gemspec
    gem install mathematical-1.7.1.gem


Other packages may be needed:
    bison
    cmake
    curl
    flex
    git
    default-jre-headless (not on macos)
    libcairo2 (macos: cairo)
    libgdk-pixbuf-2.0-0 (not on macos)
    libpango-1.0-0 (macos: pango)
    libpangocairo-1.0-0 (not on macos)
    libpangoft2-1.0-0
    libxml2 (in some cases libxml2-16)
    libglib2.0-0 (macos: glib)
    libgif7 (not on macos)
    libwebp7 (webp)
    libzstd1 (macos: zstd)
    libjpeg-turbo8 (not on macos)
    librsvg2-2 (not on macos)
    libxft2 (not on macos)
